História dos computadores

Voltar

Nessa aula foi mostrado como surgiram os primeiros computadores e sua evolução ao longo dos anos, sendo alguns dos primeiros modelos o ENIAC que era imenso e consumia grande quantidade de energia. Com o passar do tempo os computadores foram ficando bem menores, mais eficientes e portáteis.

O que é client

Voltar

Na arquitetura cliente/servidor, o client ou cliente se refere aos dispositivos que fazem alguma requisição ao servidor, explicando de forma mais simples, são os dispositivos dos usuários que acessam os serviços ou páginas da internet.

O que é server

Voltar

Na arquitetura cliente/servidor, o server ou servidor se refere ao local onde ficam hospedados ou armazenados os arquivos (páginas de html, vídeos, imagens, bancos de dados) que são exibidos para os clientes através das solicitações recebidas.

O que são linguagens de programação

Voltar

Linguagens de programação são linguagens escritas para dar instruções a uma máquina, foram criadas para facilitar o entendimento e a utilização pelos seres humanos, pois as máquinas somente aceitam instruções por código binário (0 ou 1)
